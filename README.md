# sharad-ptl.github.io

Personal site for blog posts about Platform Engineering and related topics. Built with Jekyll and hosted on GitHub Pages. Uses the [Minima](https://github.com/jekyll/minima) theme.

**Live site:** [sharad-ptl.github.io](https://sharad-ptl.github.io)

## Prerequisites

- **Ruby** 2.7 or newer (Jekyll 4 requires Ruby 2.6+)
- **Bundler** â€” install with `gem install bundler` if not already present

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/sharad-ptl/sharad-ptl.github.io.git
   cd sharad-ptl.github.io
   ```

2. Install dependencies and run the site locally:

   ```bash
   cd docs
   bundle install
   bundle exec jekyll serve
   ```

3. Open [http://localhost:4000](http://localhost:4000) in your browser.
